# Skills & Capabilities Roadmap (Future Ideas)

Este documento describe la evolución planeada para las habilidades (Skills) de WatermelonD, buscando expandir su utilidad, inteligencia e integración con el mundo real.

## Domótica e IoT Avanzado
- [ ] **Integración Profunda con Home Assistant**:
 - No solo encender luces, sino leer estados de sensores (ej: "Si la temperatura del salón > 25º, enciende el ventilador").
 - Skill de "Escenas" complejas.
- [ ] **Soporte Nativo Zigbee/Matter**:
 - Controlar dispositivos directamente con un dongle USB sin necesidad de hubs externos.
- [ ] **Presencia Inteligente**:
 - Usar escaneo WiFi/Bluetooth para saber quién está en casa y saludar o ajustar preferencias automáticamente.

## Multimedia y Entretenimiento
- [ ] **Spotify / YouTube Music Connect**:
 - Skill para buscar y reproducir listas específicas ("Pon mi mezcla de descubrimiento").
 - Modo "Multiroom" sincronizado con otros altavoces.
- [ ] **Modo Cine**:
 - Integración con Plex/Kodi/Jellyfin.
 - "Prepara el cine": Atenúa luces, cierra persianas y enciende la TV.

## Productividad y Educación
- [ ] **Code Interpreter (Intérprete de Código)**:
 - Capacidad para escribir y ejecutar pequeños scripts de Python para resolver problemas matemáticos complejos o procesar datos que el usuario le pida.
- [ ] **Resúmenes Inteligentes**:
 - Leer correos no leídos y hacer un resumen diario.
 - Resumir noticias RSS de fuentes favoritas.
- [ ] **Modo Tutor**:
 - Skills para practicar idiomas (conversación corregida).
 - Ayuda con tareas matemáticas o de programación paso a paso.

## ️ Visión Artificial (Computer Vision)
- [ ] **"¿Dónde dejé mis llaves?"**:
 - Si hay cámaras conectadas, rastrear objetos importantes y recordar su última ubicación vista.
- [ ] **Reconocimiento de Gestos**:
 - Controlar volumen o pasar canción con gestos de la mano frente a la cámara.
- [ ] **Seguridad Activa**:
 - Detectar personas desconocidas cuando no hay nadie en casa y enviar foto a Telegram/Móvil.

## Integraciones Web
- [ ] **Búsqueda Web en Tiempo Real (Perplexity-style)**:
 - Navegar por internet para responder preguntas de actualidad con citas de fuentes.
- [ ] **Compras y Listas**:
 - Integración con APIs de supermercados o Amazon para añadir productos al carrito.
- [ ] **Trámites**:
 - Consultar estado de pedidos, vuelos o citas médicas si hay API disponible.

## ️ Sistema y Mantenimiento
- [ ] **Auto-Diagnóstico Avanzado**:
 - Skill que revisa logs de error y propone soluciones (o las aplica) automáticamente.
- [ ] **Gestor de Backups Inteligente**:
 - Copias de seguridad automáticas a Google Drive / NAS cuando el sistema está en reposo.
